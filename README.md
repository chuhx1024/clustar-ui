# clustar-ui

> A Vue.js 2.0 UI Toolkit for Web By Clustar.

## Install
```shell
npm install clustar-ui -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import Clustar from 'clustar-ui'

Vue.use(Clustar)

// or
import {
  Select,
  Button
  // ...
} from 'clustar-ui'

Vue.component(Select.name, Select)
Vue.component(Button.name, Button)
```
